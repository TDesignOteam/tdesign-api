## API

### ChatLoading Props

名称 | 类型 | 默认值 | 描述 | 必传
-- | -- | -- | -- | --
className | String | - | 类名 | N
style | Object | - | 样式，TS 类型：`React.CSSProperties` | N
animation | String | moving | 加载的状态形式。可选项：skeleton/moving/gradient/dots/circle | N
text | String | - | 加载过程展示的文字内容 | N
