## API
### Empty Props

名称 | 类型 | 默认值 | 说明 | 必传
-- | -- | -- | -- | --
action | Slot | - | 操作按钮 | N
description | String / Slot | - | 描述文字 | N
external-classes | Array | - | 组件类名，分别用于设置 组件外层类名、文本描述类名、图片类名、操作按钮类名。`['t-class', 't-class-description', 't-class-image', 't-class-actions']` | N
icon | String | - | 图标名称 | N
image | String / Slot | - | 图片地址 | N
style | String | - | 自定义组件样式 | N
